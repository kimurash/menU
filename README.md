# menU
<!-- Portfolio of android application development -->
KITの学生食堂「オルタス」での献立を組み立てるAndroidアプリです  
"you"の略語として使用される"u"を敢えて大文字にすることで、
> "あなた"にとっての理想の献立を考えて欲しい  

というコンセプトを強調したつもりです

## 仕様
料理は以下の4つのカテゴリーに分類しています
- 主食
- おかず(主菜)
- 小鉢(副菜)
- デザート

それぞれの料理に対して以下の属性が表示されます
- 料金
- カロリー
- 赤緑黄の点数

ユーザは表示された料理をタップすることで、献立に料理を追加していきます  
料理の追加に伴って上記3つの属性の合計が変化します  
追加した料理を長押しすると削除の確認ダイアログが表示されます

## ファイル構成
1. res/layoutフォルダ中の.xmlファイルに画面構成を記述
2. javaフォルダ中の.javaファイルに処理を記述
3. res/valuesフォルダ中のstrings.xmlファイルに表示文字列を記述  
   アプリを多言語に対応させたい場合、  
   別言語で記述されたstrings.xmlを所定のフォルダに入れておくと、  
   言語設定に従ってOSが自動的にstrings.xmlを切り替えてくれる
